# IELTS Practice Tests

This folder contains full-length IELTS practice tests and mock exams.

## What to Store Here

- 📋 Complete IELTS practice tests
- ⏱️ Timed mock exams
- ✅ Answer keys and band score estimates
- 📊 Score tracking sheets
- 💡 Test-taking strategies

## Suggested Organization

```
practice-tests/
├── full-tests/         # Complete IELTS tests (all 4 sections)
├── listening-tests/    # Standalone listening tests
├── reading-tests/      # Standalone reading tests
├── writing-tests/      # Standalone writing tasks
├── speaking-tests/     # Standalone speaking tests
├── answer-keys/        # Answer keys and explanations
└── score-tracking/     # Templates for tracking progress
```

## File Naming Convention

- Full tests: `ielts_practice_test_[number]_[date].pdf`
- Section tests: `[section]_practice_test_[number]_[date].pdf`
- Answer keys: `answers_test_[number]_[date].pdf`

Example: `ielts_practice_test_01_2024-01.pdf`

## Test Format

A complete IELTS test includes:
- **Listening** - 30 minutes (40 questions)
- **Reading** - 60 minutes (40 questions)
- **Writing** - 60 minutes (2 tasks)
- **Speaking** - 11-14 minutes (3 parts)

## Tips

- Take tests under exam conditions (timed, no breaks)
- Review answers thoroughly after each test
- Track your scores to monitor progress
- Focus on weak areas identified in practice tests
- Aim to complete at least 5-10 full practice tests before the exam
