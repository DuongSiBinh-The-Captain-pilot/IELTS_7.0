# IELTS Vocabulary Materials

This folder contains vocabulary resources for IELTS preparation.

## What to Store Here

- 📚 Word lists by topic
- 🔤 Academic word lists
- 💡 Collocations and phrases
- 📝 Vocabulary exercises
- 🎯 Topic-specific vocabulary

## Suggested Organization

```
vocabulary/
├── by-topic/           # Vocabulary organized by topics
│   ├── education/
│   ├── environment/
│   ├── technology/
│   ├── health/
│   └── ...
├── academic-words/     # Academic Word List (AWL)
├── collocations/       # Common collocations
├── idioms/             # Useful idioms and expressions
└── exercises/          # Vocabulary practice exercises
```

## Common IELTS Topics

- Education & Learning
- Environment & Nature
- Technology & Innovation
- Health & Lifestyle
- Work & Career
- Travel & Tourism
- Culture & Society
- Media & Communication

## File Naming Convention

- Word lists: `vocab_[topic]_[date].pdf`
- Exercises: `vocab_exercise_[topic]_[date].pdf`

Example: `vocab_environment_2024-01.pdf`

## Tips

- Focus on high-frequency academic vocabulary
- Learn words in context with example sentences
- Practice collocations (words that go together)
