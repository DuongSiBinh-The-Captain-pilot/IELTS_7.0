# IELTS Writing Materials

This folder contains resources for IELTS Writing preparation.

## What to Store Here

- ✍️ Writing Task 1 samples (graphs, charts, diagrams)
- 📝 Writing Task 2 essays (opinion, discussion, problem-solution)
- 📋 Writing templates and structures
- 🎯 Band score descriptors
- 💡 Writing strategies and tips

## Suggested Organization

```
writing/
├── task1/              # Task 1 samples and practice
│   ├── graphs/         # Bar charts, line graphs, pie charts
│   ├── diagrams/       # Process diagrams, maps
│   └── tables/         # Data tables
├── task2/              # Task 2 essays
│   ├── opinion/        # Opinion/agree-disagree essays
│   ├── discussion/     # Discussion essays
│   ├── problem-solution/ # Problem-solution essays
│   └── two-part/       # Two-part questions
├── templates/          # Writing templates and structures
├── sample-essays/      # High-scoring sample essays
└── strategies/         # Writing tips and techniques
```

## File Naming Convention

- Task 1: `writing_task1_[type]_[topic]_[date].pdf`
- Task 2: `writing_task2_[type]_[topic]_[date].pdf`
- Templates: `template_[task]_[type].pdf`

Example: `writing_task1_graph_population-growth_2024-01.pdf`

## Band Score Guidelines

Store band score descriptors and criteria for each score level (5.0-9.0).
