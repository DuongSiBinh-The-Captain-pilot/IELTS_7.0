# IELTS Grammar Materials

This folder contains grammar resources for IELTS preparation.

## What to Store Here

- 📖 Grammar rules and explanations
- ✏️ Grammar exercises and practice
- 📝 Common grammar mistakes to avoid
- 🎯 Grammar for writing and speaking
- 💡 Grammar tips for each band score

## Suggested Organization

```
grammar/
├── tenses/             # Present, past, future tenses
├── conditionals/       # If-clauses and conditionals
├── passive-voice/      # Passive constructions
├── relative-clauses/   # Who, which, that, etc.
├── articles/           # A, an, the
├── modals/             # Can, could, should, must, etc.
├── sentence-structure/ # Complex sentences, conjunctions
├── punctuation/        # Punctuation rules
└── exercises/          # Practice exercises
```

## Essential Grammar Topics for IELTS

1. **Verb Tenses** - Present perfect, past perfect, future forms
2. **Complex Sentences** - Using conjunctions, relative clauses
3. **Passive Voice** - Formal writing style
4. **Conditionals** - First, second, third conditionals
5. **Modal Verbs** - Expressing possibility, obligation, advice
6. **Subject-Verb Agreement** - Ensuring grammatical accuracy
7. **Articles** - Proper use of a, an, the
8. **Prepositions** - Common prepositional phrases

## File Naming Convention

- Rules: `grammar_[topic]_rules_[date].pdf`
- Exercises: `grammar_[topic]_exercise_[date].pdf`

Example: `grammar_conditionals_rules_2024-01.pdf`

## Band Score Focus

- Band 6.0+: Mix of simple and complex sentences
- Band 7.0+: Variety of complex structures with good accuracy
- Band 8.0+: Wide range of structures with flexibility
