# IELTS Speaking Materials

This folder contains resources for IELTS Speaking preparation.

## What to Store Here

- 🎤 Speaking topics and questions
- 💬 Sample answers for all parts
- 🎯 Part 1 (Introduction) questions
- 📖 Part 2 (Cue cards) topics
- 💭 Part 3 (Discussion) questions
- 💡 Speaking strategies and tips

## Suggested Organization

```
speaking/
├── part1/              # Part 1 questions (Introduction)
├── part2/              # Part 2 cue cards (Long turn)
├── part3/              # Part 3 questions (Discussion)
├── sample-answers/     # High-scoring sample responses
├── vocabulary/         # Topic-specific vocabulary for speaking
└── strategies/         # Speaking tips and techniques
```

## File Naming Convention

- Topics: `speaking_part[1-3]_[topic]_[date].pdf`
- Sample answers: `sample_part[1-3]_[topic]_[band-score]_[date].pdf`
- Cue cards: `cuecard_[topic]_[date].pdf`

Example: `speaking_part2_favorite-book_2024-01.pdf`

## Speaking Parts Overview

- **Part 1** (4-5 mins): Introduction and general questions
- **Part 2** (3-4 mins): Individual long turn with cue card
- **Part 3** (4-5 mins): Two-way discussion

## Tips

Store recordings of practice sessions if available (with permission).
