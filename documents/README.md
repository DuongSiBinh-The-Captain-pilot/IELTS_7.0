# IELTS Documents

This directory contains all IELTS learning materials and resources organized by category.

## Directory Structure

### 📚 Core Skills
- **[listening/](listening/)** - IELTS Listening practice materials, audio files, and transcripts
- **[reading/](reading/)** - IELTS Reading passages, articles, and comprehension exercises
- **[writing/](writing/)** - IELTS Writing Task 1 & 2 samples, templates, and guides
- **[speaking/](speaking/)** - IELTS Speaking topics, sample answers, and practice questions

### 📖 Supporting Materials
- **[vocabulary/](vocabulary/)** - Word lists, collocations, and topic-specific vocabulary
- **[grammar/](grammar/)** - Grammar rules, exercises, and reference materials
- **[practice-tests/](practice-tests/)** - Full-length IELTS practice tests and mock exams

## How to Use

1. Navigate to the appropriate category folder
2. Each folder contains its own README with specific instructions
3. Add your documents following the naming conventions specified in each folder

## Contributing

When adding new materials:
- Use clear, descriptive filenames
- Include dates if materials are time-sensitive
- Add a brief description in the folder's README
- Ensure materials are properly formatted and readable
