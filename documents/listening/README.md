# IELTS Listening Materials

This folder contains resources for IELTS Listening preparation.

## What to Store Here

- 🎧 Audio files for listening practice
- 📝 Transcripts and answer keys
- 📊 Practice exercises and tests
- 📋 Note-taking templates
- 💡 Listening tips and strategies

## Suggested Organization

```
listening/
├── audio/              # MP3/WAV audio files
├── transcripts/        # Text transcripts of audio files
├── exercises/          # Practice exercises
├── answer-keys/        # Answer keys and explanations
└── strategies/         # Tips and techniques
```

## File Naming Convention

- Audio: `listening_section[1-4]_[topic]_[date].mp3`
- Transcripts: `transcript_section[1-4]_[topic]_[date].pdf`
- Exercises: `exercise_[topic]_[level]_[date].pdf`

Example: `listening_section1_rental-accommodation_2024-01.mp3`
