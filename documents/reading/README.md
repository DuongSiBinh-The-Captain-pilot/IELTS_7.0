# IELTS Reading Materials

This folder contains resources for IELTS Reading preparation.

## What to Store Here

- 📰 Reading passages and articles
- ❓ Comprehension questions
- ✅ Answer keys and explanations
- 📚 Academic and general training texts
- 💡 Reading strategies and tips

## Suggested Organization

```
reading/
├── academic/           # Academic reading passages
├── general-training/   # General training passages
├── practice-tests/     # Full reading practice tests
├── answer-keys/        # Answer keys and explanations
└── strategies/         # Tips and techniques
```

## File Naming Convention

- Passages: `reading_[type]_[topic]_[date].pdf`
- Practice tests: `reading_test_[number]_[date].pdf`
- Answer keys: `answers_reading_[test-name]_[date].pdf`

Example: `reading_academic_climate-change_2024-01.pdf`

## Tips

- Academic passages: scientific, historical, or educational topics
- General Training: workplace, social, or everyday topics
